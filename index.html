안녕하세요, 정원중입니다

이번 과제에서는 러버블 AI를 활용하여 자기소개 앱을 제작했습니다.

AI를 활용하면서 코드 작성이 훨씬 쉬워졌고, 디자인 아이디어도 참고할 수 있었습니다.

또한 오디오와 비디오 삽입 방법, HTML 기본 구조를 학습하면서 웹 개발의 기초를 익혔습니다.

이 경험을 통해 앞으로 더 복잡한 프로젝트에도 도전할 수 있을 것 같습니다.
